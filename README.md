# 🚗 RideBook Interface – Uber-Style Frontend Platform

**RideBook Interface** is a fully responsive, multi-language, frontend-only ride-booking platform inspired by Uber.
It delivers a complete user experience with clean UI, smooth animations, and a modern component-based architecture.

---

# ✨ Features

## 🔐 Authentication

* Animated login page
* “Dive In” quick entry (no backend authentication)
* Sign-out functionality

---

## 🏠 Home Page

* Pickup / Dropoff booking form
* Uber-style creative map animation
* 3-step booking guide
* 6 ride-service cards
* Full multilingual user interface

---

## 🌐 Supported Languages (8)

* English
* Bengali
* Hindi
* Kannada
* Marathi
* Tamil
* Telugu
* Urdu

# 🚧 Implementation

* React Context used for language state management
* State-based navigation without React Router
* Lazy-loaded images for performance optimization
* Vite optimized production builds
* Semantic HTML with ARIA accessibility labels
